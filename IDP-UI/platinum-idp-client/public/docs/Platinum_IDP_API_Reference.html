1. Cycles

IDP cycle lifecycle management (Section 5.1). Status flow: Draft → In Review → Approved for Distribution → Adopted → Revised

GET /api/cycles
List all IDP cycles, ordered by start year descending.
Returns: IdpCycle[]
GET /api/cycles/{id}
Get a single cycle by ID.

Path Parameters

NameTypeDescription
idintCycle ID
Returns: IdpCycle
POST /api/cycles
Create a new IDP cycle. An audit log entry is created automatically.

Request Body

FieldTypeRequired
namestringRequired
startYearintRequired
endYearintRequired
municipalityNamestringRequired
statusstringOptional (default: Draft)
revisionNumberintOptional (default: 1)
descriptionstring?Optional
isLockedboolOptional (default: false)
Returns: IdpCycle (201 Created)
PUT /api/cycles/{id}
Update an existing cycle. The ID in the body must match the URL parameter.

Path Parameters

NameTypeDescription
idintCycle ID
Returns: 204 No Content
PATCH /api/cycles/{id}/status
Change cycle status. Valid transitions: Draft → In Review → Approved for Distribution → Adopted → Revised. An audit log entry is created.

Request Body

FieldTypeDescription
statusstringNew status value
Returns: 204 No Content
GET /api/cycles/{id}/dashboard
Get comprehensive dashboard statistics for a cycle: project counts, comment stats, milestone progress, budget totals, phases, document/submission counts, and recent audit logs.
Returns: DashboardData

14. Priority Scoring

Score projects against framework criteria, calculate composites, get rankings, and run budget simulations.

GET /api/priority-scores/framework/{fwId}/project/{projId}
Get all scores for a specific project within a framework.
Returns: PriorityProjectScore[]
GET /api/priority-scores/framework/{fwId}/all
Get all project scores and rankings for a framework.
Returns: ProjectRanking[]
POST /api/priority-scores/score
Score a single project on a single criterion. Creates or updates the score.

Request Body

FieldTypeRequired
frameworkIdintRequired
projectIdintRequired
criteriaIdintRequired
humanScoreint?Optional
Returns: PriorityProjectScore
POST /api/priority-scores/framework/{fwId}/project/{projId}/score-all
Batch score all criteria for a project at once.

Request Body

FieldType
Array of { criteriaId: int, humanScore: int }ScoreBatchItem[]
Returns: 204 No Content
GET /api/priority-scores/framework/{fwId}/project/{projId}/composite
Calculate composite score for a project. Formula: sum(criteriaWeight x blendedScore) / 100.
Returns: { compositeScore: decimal }
POST /api/priority-scores/framework/{fwId}/ai-recommend/{projId}
Generate AI-recommended scores for a project based on project attributes. Uses rule-based scoring engine.
Returns: PriorityProjectScore[]
POST /api/priority-scores/framework/{fwId}/ai-recommend-all
Generate AI-recommended scores for ALL projects in the framework's linked cycle.
Returns: { projectsScored: int }
GET /api/priority-scores/framework/{fwId}/rankings
Get ranked project list ordered by composite score (or override rank if set).
Returns: ProjectRanking[] (sorted by rank)
POST /api/priority-scores/framework/{fwId}/save-ranks
Save manual rank overrides for projects. Persists to IdpProject.OverrideRank and updates PriorityRanking.

Request Body

FieldType
Array of { projectId: int, rank: int }RankOverride[]
Returns: 204 No Content
GET /api/priority-scores/framework/{fwId}/budget-simulation
Run budget impact simulation. Returns ranked projects with running totals and funding cut-off analysis.

Query Parameters

NameTypeDescription
thresholddecimal?Budget ceiling for cut-off analysis
Returns: BudgetSimulationResult
